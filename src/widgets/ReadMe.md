Тут должны лежать только виджеты и layout(ы)

виджеты могут импортировать в себя: 
  features, 
  entity, 
  shared(этого уровня у тебя скорее всего не будет)

виджеты НЕ МОГУТ импортировать:
  другие виджеты,
  страницы (_pages)
  приложения (app)

например: 
  виджет авторизации
  виджет таблицы для данных пользователя
